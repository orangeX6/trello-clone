Certainly! Here's the updated code using Tailwind CSS classes to display locations vertically with borders, padding, and additional styling:

jsx
import { useFetchLocationsQuery } from '../store';
import { LocationType } from '../types/location-type';

export const LocationList = () => {
  const { data, error, isFetching } = useFetchLocationsQuery({ tenant: import.meta.env.VITE_TENANT });

  let locations;
  if (data) {
    locations = data.locationList.resources.map((el: LocationType) => (
      <div className='border rounded p-4 mb-4' id={el.id}>
        <p className='text-lg font-bold'>Title: {el.name}</p>
      </div>
    ));
  }

  return <div>{locations}</div>;
};

In this code, I've used various Tailwind CSS classes to style the location items:

  • border adds a border around each location item.
  • rounded gives the items rounded corners.
  • p-4 adds padding of 4 units on all sides.
  • mb-4 adds a margin-bottom of 4 units to create space between the location items.
  • text-lg sets the font size to large.
  • font-bold makes the text bold.

Make sure to have Tailwind CSS properly set up in your project, and the classes mentioned above should work as expected. Feel free to adjust the classes and their values to match your specific styling preferences.
